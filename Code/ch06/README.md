## 6.27

![1717049672047](https://gitee.com/porlain/clouding/raw/master/202405301426139.jpg)

A.
标记位为 45 时：
CT = 01000101 CI = 001 CO = XX
因此地址为 0x08A4 ~ 0x08A7

标记位为 38 时：
CT = 00111000 CI = 001 CO = XX
因此地址为 0x0704 ~ 0x0707

B.
0x1238 ~ 0x123B
